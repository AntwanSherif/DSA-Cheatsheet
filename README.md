# DSA Cheatsheet

A single-file interview prep reference covering 17 core algorithm patterns at easy/medium LeetCode difficulty.

## Usage

Open `dsa-cheatsheet.html` in any browser. No build step, no dependencies beyond a CDN connection for Mermaid diagrams and syntax highlighting.

## Algorithms Covered

| Category | Topics |
|---|---|
| Arrays | Two Pointers · Sliding Window · Prefix Sums |
| Hashing | Hash Maps & Sets |
| Linked Lists | Fast & Slow Pointers · Linked List Reversal |
| Trees | DFS · BFS · BST |
| Graphs | BFS · DFS |
| Search & Sort | Binary Search · Sorting |
| Stack | Monotonic Stack |
| Dynamic Programming | 1D DP · 2D DP |
| Backtracking | Subsets / Permutations / Combinations |

## Each tab includes

- When to reach for this pattern (recognition cues)
- Visual diagram (Mermaid with colors or ASCII)
- Step-by-step walkthrough
- TypeScript code template
- 2–4 LeetCode practice problems with key insights
